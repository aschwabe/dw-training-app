by Andrew Schwabe <aschwabe@gmail.com>
Twitter: @aschwabe

Notes:

- Maven Java Project
- Uses Dropwizard Framework
- Uses Boxfuse for Containerization and Deployment
- Uses Shippable for Continuous Integration/Continuous Delivery


How to get started:

1. Sign up for github or bitbucket (git) - FREE
2. Sign up for boxfuse - FREE
3. Sign up for shippable - FREE
4. Download Eclipse - FREE
5. Import this project into Eclipse
6. Fork or create git repository for the code
7. Edit pom.xml and put your boxfuse credentials in
8. In Shippable, connect to your git repository and enable CI
9. Edit shippable.xml to use the right project names and paths etc.
